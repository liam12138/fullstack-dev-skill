# 全栈开发工程师 Skill

## 功能特性

### 项目初始化
- 项目类型选择（Web、移动端、后端、桌面端、CLI）
- 项目名称和描述收集
- 初始配置生成

### 需求收集
- 功能需求交互问答
- 非功能需求收集
- 约束条件分析
- 自动生成需求文档

### 技术方案生成
- 智能技术栈推荐
- 架构设计（单体、前后端分离、微服务）
- 多应用技术方案生成
- 技术决策对话确认

### 代码生成
- 项目结构自动生成
- 核心代码骨架
- 配置文件生成
- 支持多种技术栈

### 对话确认与回退
- 每一步都支持确认修改
- 支持回退到上一步
- 随时重置流程

## 技术栈支持

- React + TypeScript + Vite
- Vue 3 + TypeScript + Vite
- Node.js + Express + TypeScript
- Python + FastAPI
- Java + Spring Boot

## 触发命令

- `/init` - 开始项目初始化
- `/new-project` - 创建新项目
- `/fullstack` - 启动全栈开发流程

## 触发短语

- "帮我初始化一个新项目"
- "创建一个新项目"
- "开始一个新项目"
- "帮我设计技术方案"
- "生成项目代码"

## 工作流程

1. **项目初始化** - 收集项目基本信息
2. **需求收集** - 通过对话收集功能和非功能需求
3. **技术方案** - 生成技术方案文档并确认
4. **代码生成** - 根据技术方案生成初始代码结构
5. **项目交付** - 提供后续开发指导

## 使用说明

### 快速开始

1. 在 Trae 或 Claude Code 中激活 Skill
2. 输入 `/init` 或相关触发短语开始流程
3. 按照提示回答问题
4. 每一步可使用 `确认`、`回退` 或 `重置` 命令

### 交互命令

- `确认` - 确认当前步骤并进入下一步
- `回退` - 返回上一步进行修改
- `重置` - 重置整个流程重新开始
- `跳过` - 跳过当前步骤（如适用）
